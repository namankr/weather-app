Getting started !!

Prereqisite to run this project node installed if not please install from - https://nodejs.org/

Install mongo db client server - https://docs.mongodb.com/v3.2/tutorial/install-mongodb-on-windows/

Running server - 

1. open the terminal under folder node-mongo-registration-login-api 

Type below two commands 

2. npm install 
3. npm start 


This should start the server at port 4000


Running the to run the pages

1. open the terminal under folder website

Type below two commands 

2. npm install 
3. npm start 


This should start serving the pages at localhost 


Notes :
 
This is still not a production ready code/project and still some 
1. bugs need to be fixed.
2. UI Can be improved a lot.
